# Task Management System

Bu proje, kullanıcıların günlük işlerini, projelerini ve hedeflerini organize etmelerine yardımcı olan bir **Kişisel Görev Yönetim Sistemi** web uygulamasıdır.

**Geliştirici:** Doğukan Oğan

## Kullanılan Teknolojiler

- **Backend:** .NET 8 Web API
- **Frontend:** Angular 17+
- **Veritabanı İşlemleri:** Entity Framework Core
- **Ana Veritabanı:** PostgreSQL
- **Alternatif Veritabanı:** Oracle
- **Kimlik Doğrulama:** JWT (JSON Web Token)

## Proje Yapısı

- `Backend/` - .NET 8 Web API projesini ve Entity modellerini içerir.
- `Frontend/` - Angular tabanlı kullanıcı arayüzünü içerir.
- `Database/` - Referans amaçlı SQL şema ve örnek veri betiklerini barındırır.
